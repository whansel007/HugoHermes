# MLOps Umbrella

This skill serves as the central hub for machine learning operations, including tracking, benchmarking, and serving.

## Components
- **Weights & Biases (W&B):** Experiment tracking and model registry.
- **Evaluation:** Benchmarking with lm-eval-harness.
- **Deployment & Serving:**
    - vLLM (high-throughput serving)
    - llama.cpp (local GGUF inference)
    - HuggingFace Hub (model/dataset management)
    - Segment Anything (SAM)
- **Model Surgery:** OBLITERATUS (abliterating model refusals).
